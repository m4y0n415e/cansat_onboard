\doxysubsection{Detailed Description}
Software RTC with GPS synchronization and Fat\+FS integration. 

Maintains system time using system ticks ({\ttfamily to\+\_\+ms\+\_\+since\+\_\+boot}), independent of hardware RTC. It features:\+
\begin{DoxyItemize}
\item a sync with GPS:\+ corrects drift and sets time/\+date via NMEA data.
\item a Fat\+FS backend:\+ provides timestamps for SD card files.
\item timezones:\+ applies static offsets defined by {\ttfamily \doxylink{time__manager_8c_ad002a29d99eafe389580285df1b17b62}{TIMEZONE\+\_\+\+OFFSET}}.
\item Usage:\+ Calling {\ttfamily \doxylink{time__manager_8h_a3a22b1d94e247980fd1a29ae8eb19bf6}{time\+\_\+manager\+\_\+init()}} at startup, {\ttfamily \doxylink{time__manager_8h_a7f053b33e4a11799b757855ab7a4d6ea}{time\+\_\+manager\+\_\+update()}} periodically in the loop, and {\ttfamily \doxylink{time__manager_8h_a31a781682c9a9732cb9109c8cd7300ac}{time\+\_\+manager\+\_\+sync()}} when valid GPS data exists.
\item \begin{DoxyNote}{Note}
Replaces the default {\ttfamily rtc.\+c} to remove hardware dependencies that were trublesome. 
\end{DoxyNote}

\end{DoxyItemize}

Ticks the internal clock forward. 

This function checks the system\textquotesingle{}s millisecond timer. If 1000ms have passed since the last tick, it increments the internal {\ttfamily raw\+\_\+seconds} counter and updates the reference timer. This function is non-\/blocking and is designed to be called frequently inside the main {\ttfamily while(1)} loop.
