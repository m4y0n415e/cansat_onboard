\doxysubsection{Detailed Description}
Atmospheric sensors module interface. 

This file handles the configuration of all atmospheric sensors, through I2C, SPI and UART. It provides functions to initialize and conduct readings from all the sensors, saving them to a dedicated \textquotesingle{}\doxylink{structsensor__readings__t}{sensor\+\_\+readings\+\_\+t}\textquotesingle{} structure.
